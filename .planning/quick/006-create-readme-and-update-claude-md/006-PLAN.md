# Quick Task 006: Create README.md and Update CLAUDE.md

## Goal

Create a comprehensive README.md and update CLAUDE.md with best practices based on
research.

## Tasks

### Task 1: Create README.md

Create README.md with:

- Project name and description
- Features overview
- Tech stack
- Getting started (prerequisites, installation, running locally)
- Project structure
- Contributing guidelines
- License/disclaimer

### Task 2: Update CLAUDE.md

Rewrite CLAUDE.md following best practices:

- Keep under 100 lines
- WHAT: Tech stack, project structure map
- WHY: Project purpose and core value
- HOW: Terminal commands, build process, testing
- Include workflows for common tasks
- Reference .planning/ docs for deeper context
- Avoid code style rules (use Biome)
